### The Problem I'm Solving Now

Research predicts outcomes. But **predictions don't save patients â€” systems do.**

Every day, clinicians spend **2+ hours on documentation** instead of patients. AI agents could automate this â€” but in healthcare, an unaudited AI decision can harm someone, violate HIPAA, or fail in court.

**The industry has a trust gap.** AI frameworks build agents. Governance frameworks write policy documents. Nobody enforces policy *at runtime*, with cryptographic evidence, on every single agent action.

That's what I build now.
